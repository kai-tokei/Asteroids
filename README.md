## [操作]
旋回: 左右キー  
前進: 上キー  
発射: スペース  

## [仕様言語など]
JavaScript  
p5.js  

## [こだわり]
多角形を描画する関数の自作  
宇宙らしい慣性の再現  

## [技術面]
当たり判定(ピタゴラスの定理を用いた、斜辺の長さから重なっているかを判別するシンプルな判定)  
多角形の描画(配列にラジアンの角度を代入, 線を引いている)  
慣性(運動方程式を使用, 0.98の値で自然に速度減衰している)
